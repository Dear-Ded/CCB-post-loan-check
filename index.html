Public intelligence / 公开情报

统一接入公开网页、官方平台、搜索结果和用户授权数据入口，不生产数据，不伪造结论。

  • Source registry | 数据源目录与准入状态
  • Health checks | 连通性、可用性、失败原因
  • Provenance | URL、时间、截图、审计痕迹

Subject profile / 主体画像

从企业延伸到实控人、产品、行业、资本和关联主体，形成可解释的风险认知。

  • Controller linking | 法人/实控人关联
  • Bounded fan-out | 可配置递归深度
  • Evidence grading | 事实、推断、待核验分层

Risk discovery / 风险发现

目标不是写更长的报告，而是帮助用户发现原本可能被遗漏的经营、信用和治理风险。

  • Financial clues | 财务异常线索引擎
  • Industry context | 行业与产品生命周期
  • Signal trigger | 事件驱动式升级分析

Codex adapter

面向本地插件和 skill 运行，强调自动执行、证据留痕、报告产出和可恢复任务。

  • Plugin metadata aligned to 0.5.0
  • Skill package keeps runtime compatibility

WorkBuddy package

把复杂的运行参数压成表单，让非技术用户输入企业名就能跑。

  • Plain-language preflight
  • Expert package and manifest ready

Doubao task mode

适配 PC、本地浏览器、远程浏览器和 App 办公任务模式，输出契约保持一致。

  • Task-mode metadata cleaned
  • Batch reports and evidence folders aligned
