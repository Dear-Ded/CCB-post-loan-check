# 豆包任务提示模板

用户可直接说：

```text
帮我做 CCB贷前贷后查询，企业名是 XXX，生成 Word 报告。
```

批量：

```text
帮我批量做 CCB贷前贷后查询：企业A、企业B、企业C。结果只给我 reports 文件夹。
```

## 执行器硬要求

- 不得模拟。
- 不得墨迹。
- 不能胡编乱造数据。
- 必须原原本本执行项目脚本。
- 自动填写企业名、统一社会信用代码或组织机构代码、身份证号等非挑战字段。
- 页面挑战确认失败时保持页面，等待真实结果或记录缺口。
- 未确认真实结果/无结果前不得截图。
- 不得把登录页、挑战页、异常页、空白页写入最终 Word。
